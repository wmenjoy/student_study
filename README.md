# 小学生学习应用 (Elementary Student Learning App)

一个专为小学生设计的互动式数学学习平台，通过游戏化的方式帮助学生掌握数学概念和解题技巧。

## 📚 项目简介

这是一个基于 Next.js 和 TypeScript 构建的教育应用，提供多种互动式数学学习模块，包括：

- **数学爬梯游戏** - 通过闯关方式练习数学题目
- **逻辑推理训练** - 如次品口香糖问题的逻辑树分析
- **可视化演示** - 包括等差数列求和、移多补少等数学概念的动画演示
- **分级学习内容** - 针对1-6年级不同难度的题目和知识点

## 🚀 技术栈

- **框架**: Next.js 14.2
- **语言**: TypeScript 5.6
- **UI**: React 18.2
- **动画**: Framer Motion 11.0
- **状态管理**: Zustand 4.5, XState 5.17
- **数据存储**: Dexie 4.0 (IndexedDB)
- **测试**: Vitest 1.6

## 📦 安装与运行

### 前置要求

- Node.js 18+ 
- npm 或 yarn

### 安装依赖

```bash
cd apps/student
npm install
```

### 开发模式

```bash
npm run dev
```

应用将在 [http://localhost:3000](http://localhost:3000) 启动

### 构建生产版本

```bash
npm run build
npm start
```

### 运行测试

```bash
npm test
```

## 🎮 主要功能模块

### 1. 数学爬梯游戏 (Math Climbing)
- 分级难度系统（1-6年级）
- 涵盖加减乘除、分数、小数等多种题型
- 进度追踪和成就系统

### 2. 逻辑推理训练
- 次品口香糖问题
- 逻辑树可视化
- 天平模拟演示

### 3. 数学概念可视化
- 等差数列求和（高斯方法）
- 移多补少问题
- 植树问题
- 爬楼梯问题

## 🎨 设计理念

- **游戏化学习**: 通过互动游戏提高学习兴趣
- **可视化教学**: 用动画和图形帮助理解抽象概念
- **个性化进度**: 根据年级和能力调整难度
- **即时反馈**: 实时的答题反馈和鼓励

## 🛠️ 开发指南

### 添加新的课程模块

1. 在 `components/` 下创建新组件
2. 在 `lib/catalog.ts` 中注册课程
3. 在 `app/` 下创建对应路由

### 代码规范

- 使用 TypeScript 严格模式
- 遵循 ESLint 配置
- 组件使用函数式编程
- 优先使用 React Hooks
