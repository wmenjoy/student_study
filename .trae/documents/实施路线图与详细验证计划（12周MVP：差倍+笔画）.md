# 总体节奏与里程碑（12周）
- W0–1 准备与基线：团队到位、技术选型、设计规范、内容框架与数据策略确定
- W1–2 教具与关卡骨架：比例条/数轴/拼板组件、关卡DSL与运行时、笔画演示原型
- W3–6 核心专题与家长端：差倍完整变式训练、笔画纠错、词语小游戏、家长简版报表
- W7–9 自适应与错因诊断：掌握度模型、动态出题、错因识别与微指导、作品墙、离线同步
- W10–12 打磨与验证：可及性完善、性能与稳定性、学习成效验证、A/B试验与发布准备

# 实施阶段与任务分解
- 阶段0：准备与基线（W0–1）
  - 目标：建立统一规范与最小可运行架构
  - 任务：
    - 设计规范与UI套件（字号、色彩、动效、儿童友好控件）
    - 内容框架：题型元数据、关卡DSL（场景、脚本、提示、评估）
    - 数据策略：本地存储结构（IndexedDB/SQLite）与增量同步协议
    - 可观测性：事件埋点方案与核心KPI定义（掌握度、参与度、体验）
  - 交付：系统骨架、设计系统v1、数据字典、埋点白名单

- 阶段1：交互教具与关卡骨架（W1–2）
  - 目标：搭建复用的可操作教具与关卡运行时
  - 任务：
    - 比例条组件：长度比例、标签、差值标注、吸附对齐、步骤记录
    - 数轴组件：刻度、区间、跳跃步长、标记拖拽与对齐
    - 几何拼板：拼接、旋转、面积/周长自动计算
    - 关卡运行时：剧情、引导提示、错误回路、微测验触发
  - 交付：教具库v1、关卡引擎v1、示例关卡1（差倍引题）

- 阶段2：核心专题与家长端（W3–6）
  - 目标：差倍与笔画的完整可学闭环，家长侧可视化
  - 任务：
    - 差倍专题：图解建模→图式映射→等价变式→微测→复练
    - 笔画演示与跟写：笔顺动画、轨迹采集、匹配评分、即时纠错与部件高亮
    - 词语小游戏：近反义词、成语接龙（轻量），即时反馈与例句纠错
    - 家长简版：技能雷达、错因Top3、学习周报（自动生成）
  - 交付：差倍专题v1、笔画练习v1、家长报表v1

- 阶段3：自适应与错因诊断、作品与离线（W7–9）
  - 目标：形成个性化学习循环与作品激励，保障离线体验
  - 任务：
    - 掌握度模型：初始化、更新与持久化（简版BKT或ELO）
    - 动态出题：按掌握度与错因驱动题型与难度选择，题面多样化
    - 错因识别：≥3类常见错因（差倍混淆、单位遗漏、式子构造错误）绑定微指导脚本
    - 作品墙：保存图解/拼板/故事作品，允许注释与分享
    - 离线与同步：断网学习与联网增量合并，冲突提示与解决
  - 交付：自适应服务v1、错因引擎v1、作品墙v1、离线同步v1

- 阶段4：可及性与性能打磨、验证收尾（W10–12）
  - 目标：达成上线标准与学习成效指标
  - 任务：
    - 可及性：TTS旁白、拼音辅助、色盲主题、可调字体、键盘与触控友好
    - 性能与稳定：交互延迟<100ms、离线加载<2s、错误率控制与降级策略
    - 学习成效验证：微测与迁移题评估、A/B试验、周报与家长反馈闭环
    - 发布与回滚：灰度策略、日志与告警、快速回滚方案
  - 交付：可及性v1、性能达标、验证报告、发布包与回滚文档

# 团队分工与协作
- 产品：PRD、验收标准、指标方案与实验设计
- 教研：题型设计、变式质量、错因脚本与微指导
- 前端：UI与交互教具、关卡运行时、家长端
- 算法/后端：掌握度模型、动态出题、同步服务与可观测性
- QA/数据：测试体系、数据管线与指标看板

# 验证计划总览（层级与方法）
- 单元测试：组件逻辑（比例条、数轴、笔画匹配、掌握度更新）覆盖率≥80%
- 集成测试：关卡流程（引题→建模→图式→变式→微测）与错因触发闭环
- E2E测试：核心路径的真实交互（含触控），跨设备与离线场景
- 可用性测试：与10–20名儿童进行可用性观察与任务完成率评估
- 学习成效：前后测（掌握度与迁移题正确率）、AB实验对照
- 性能与可靠性：交互延迟、加载时间、崩溃率、离线稳定性、同步冲突
- 安全与隐私：数据最小化、导出/删除、家长监护与合规检查
- 可及性：TTS有效覆盖、拼音正确性、色盲主题对比度、键盘导航

# 详细验收与测试用例矩阵
- EPIC A：差倍问题
  - A1 图解建模：
    - 用例：拖拽生成比例条、标注基准量、标注差距
    - 断言：系统正确识别“谁是倍数/差值”，错误触发微指导
  - A2 图→式映射：
    - 用例：从比例条生成口语解释与等式
    - 断言：表达一致性与等式正确性（≥95%题面）
  - A3 等价变式：
    - 用例：已知差求倍、已知倍求差、两量同时变动
    - 断言：每题型≥3道，掌握度关联的难度调节生效
  - A4 微测与掌握度更新：
    - 用例：微测题通过率≥80%标记掌握
    - 断言：不达标进入补练，补练后通过率提升≥20%

- EPIC B：笔画练习
  - B1 笔顺演示与跟写：
    - 用例：播放笔顺动画、采集轨迹并评分
    - 断言：匹配度≥70%判定通过，边界与异常轨迹稳定
  - B2 即时纠错与部件高亮：
    - 用例：识别起笔/收笔/方向/顺序错误，高亮偏旁
    - 断言：≥80%错误识别准确，提示不打断流
  - B3 词语小游戏：
    - 用例：近反义词、成语接龙
    - 断言：即时反馈、误用给例句纠正

- EPIC C：教具模板
  - C1 比例条：吸附对齐与标签一致性
  - C2 数轴：刻度准确、跳跃步长正确
  - C3 拼板：面积/周长计算正确性与浮点稳定

- EPIC D：自适应与错因诊断
  - D1 掌握度模型：初始化与更新的单元与集成测试
  - D2 动态出题：输入掌握度与错因，输出题型与难度符合策略
  - D3 错因识别：≥3类错因与脚本绑定的覆盖率测试

- EPIC E：正反馈与作品
  - E1 进度地图与徽章触发：达成条件准确、重复触发去重
  - E2 作品墙：保存、复盘注释与分享流程完整

- EPIC F：家长/老师端
  - F1 技能雷达与错因摘要：数据一致性与可读性
  - F2 周报与任务布置：自动生成与任务包下发成功率

- EPIC G：离线与同步
  - G1 本地存储：断网可运行且不丢进度
  - G2 增量同步：联网后合并成功，无冲突或可提示解决

- EPIC H：可及性
  - H1 TTS与拼音：覆盖率与正确性测试
  - H2 色盲主题与字体：对比度与可读性门槛达标

# 指标与质量门槛（Go/No-Go）
- 学习：差倍掌握度≥0.8的孩子占比≥70%；迁移正确率≥60%
- 参与：7日留存≥30%，平均会话≥12分钟，创作≥1/周
- 体验：家长满意度≥4/5，交互延迟<100ms，离线加载<2s
- 稳定：崩溃率≤行业均值，错误率<1%，同步冲突可解率≥95%
- 合规与可及性：隐私与监护合规全部通过；TTS覆盖100%，色盲主题与键盘导航达标

# 数据与可观测性
- 埋点字典：关卡事件（进入/完成/失败）、教具操作（拖拽/标注）、错因触发、微测结果、作品保存、家长端可视化访问
- 指标看板：掌握度趋势、错因Top、参与度与留存、性能与稳定性、A/B实验结果

# 风险与应对
- 题海化：以变式质量与建模深度为核心，限制重复低质量练习
- 外在奖励依赖：创作与探索驱动，奖励轻量化、不强绑定
- 技术复杂度：模板优先，AI能力渐进接入，关键路径先打通

# 发布与回滚
- 灰度与白名单：先小范围用户，监测指标门槛达标再扩大
- 回滚策略：功能开关与版本回退，数据迁移可逆性验证

# 需确认
- 目标平台优先顺序（Web/移动/平板）与终端适配范围
- 家长端最小可用范围与周报周期（周/月）
- 儿童可用性测试样本量与伦理审批流程
